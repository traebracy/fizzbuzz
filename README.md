# FizzBuzz

A small JavaScript practice project for the classic FizzBuzz programming exercise.

## Purpose

This repository is kept as a simple fundamentals exercise. It focuses on conditional logic, loops, and basic JavaScript problem solving.

## Core Logic

The program checks a range of numbers and prints:

- `Fizz` for numbers divisible by 3
- `Buzz` for numbers divisible by 5
- `FizzBuzz` for numbers divisible by both 3 and 5
- The original number when none of those conditions apply
